# 🟣 Data & Insights

<figure><img src="../../../.gitbook/assets/AoM Section_Divider_Purple.png" alt="AoM Data &#x26; Insights section banner, five line-art sensory icons, an eye, a nose, a hand, an ear, and a mouth, arranged in the layout of a face, on a purple background."><figcaption></figcaption></figure>

### What is it?

Observe, interpret, and apply insights to inform better decisions.

### Why it matters

Like the senses, this Part informs decisions through observation and feedback. It allows the organisation to learn from what is happening in the market. When this part is strong, decisions improve over time based on real signals. When this part is weak, decisions are made without context, and the organisation continues to act without seeing, hearing, or learning from what is actually happening.
